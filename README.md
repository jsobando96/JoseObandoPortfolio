## Projects

### Quirch Foods
WordPress development for an international food brand based in the U.S. Multi-site support across mambofoods.com, quirchfoods.com, panamei.com, and quirch.com/chiquita — covering responsive page development, SEO-conscious structure, and long-term maintenance over 4 years.

### Pilar Velez
Publishing platform for a Hispanic heritage literature organization. Custom WordPress ecosystem across four websites with ACF-driven content systems, custom taxonomies, search and filtering, and editor-friendly admin workflows for non-technical teams.

### Pharmabrand
A portfolio series of three pharmaceutical campaign builds:
- **Fast-turnaround landing pages** — luvis.ec, binmune.ec, nocol.ec, biofit.com.ec
- **Guardianes de la Salud** — Medical code redemption platform with 7,000+ code submissions and 2,000+ registered doctors
- **OftalmoFoto** — Photography contest platform with registration and backend image management

### Pukllay
Custom WordPress + WooCommerce registration system for a sports events company in Ecuador. Dynamic multi-participant forms, conditional pricing for jerseys and race bundles, automated event setup, and admin workflows — supporting 25+ running and cycling events.
